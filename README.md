## Available Scripts

To run the App 
1. go inside the *'cmch_front'* and run *`npm install`* then `npm start`
2. go inside the *'cmch_back'* and run *`npm install`* then `npm start`
3. The App will be runing in https://localhost:3000

### `nedded Tools To start The App`

- nodeJs
- MongoDB
